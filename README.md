# React Crypto Initial

Начальный проект для криптовалютного приложения.

## О проекте
Это стартовый шаблон full-stack приложения, состоящий из:
*   **Frontend**: React-приложение (папка `frontend`)
*   **Backend**: Node.js сервер (`server.js`)

Проект находится на начальной стадии разработки.

## Технологии
*   JavaScript
*   React
*   Node.js

## Быстрый старт
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/D1mkaaa/react-crypto-initial.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd react-crypto-initial
    ```
3.  Установите зависимости:
    ```bash
    npm install
    ```
4.  Запустите сервер (команду уточните в `package.json`):
    ```bash
    npm start
    ```
